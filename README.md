# 个人状态记录器

一个面向个人使用、移动端优先的轻量状态记录工具。  
目标不是“做复杂分析”，而是让你能在真实生活里低阻力地留下状态痕迹，并逐渐看到变化趋势。

当前项目已经从最初的基础记录工具，迭代到了支持：

- 每日状态记录
- 快速记录与低阻力记录
- 最近 30 天任意日期补记
- Dashboard 最近 7 天记录闭环
- 历史记录 / 趋势图
- JSON 导入 / 导出
- PWA 安装到手机主屏幕
- LocalStorage 本地存储

---

## 1. 项目定位

「个人状态记录器」是一个个人使用的状态观察工具，核心目标是：

- 用尽量低的成本完成记录
- 让记录在状态差的时候也能发生
- 用趋势和历史帮助自己从“模糊感受”进入“可观察状态”

它不是：

- 周复盘系统
- AI 分析工具
- 标签 / 日历 / 打卡系统
- 云同步产品
- 社交产品
- 心理诊断工具

## 2. 当前能力概览

## 7. 数据存储说明

应用数据保存在当前设备浏览器的 LocalStorage 中，key 为：

```text
personal_state_records
```

说明：
- 不会上传到服务器
- 不会自动同步到其他设备
- 清理浏览器缓存后数据可能丢失
- 建议定期导出 JSON 备份

## 8. 数据迁移

如果你想把电脑上的数据迁移到手机：

1. 在电脑端打开应用；
2. 进入设置页，点击“导出 JSON”；
3. 将导出的 JSON 文件发送到手机；
4. 在手机端打开部署后的应用；
5. 进入设置页，点击“导入 JSON”；
6. 选择文件完成导入。

导入规则：
- 顶层必须是 `StateRecord[]`
- 以 `date` 作为唯一标识
- 同日期记录会覆盖本地记录
- 新日期记录会新增
- 非法 JSON 不会覆盖现有本地数据

## 9. 当前记录体验

### 快速记录
记录页默认走低阻力路径：

- 选日期
- 点几个接近今天状态的片段
- 直接保存

### 状态片段
当前快速记录按分组组织，例如：
- 身体电量
- 事情推进
- 人际连接
- 脑内状态

每组内单选，分组之间可以组合。

### 详细分数
默认折叠。  
只有在你想更精细记录时才展开 6 个 Slider。

### 补充记录
默认折叠。  
不写文字也可以，先留下状态就够了。

## 10. Dashboard 闭环

Dashboard 不只是看状态，还承担“提醒并引导记录”的作用。

当前支持：

- 今日是否已记录
- 最近 7 天覆盖率
- 最近 7 天记录概览
- 点击某一天直接进入对应日期记录页
- 今日未记录时快速入口
- 今日已记录时快速编辑入口

这使得记录路径更接近：

```text
打开应用 → 看到最近 7 天 → 点某一天 → 直接补记或编辑
```

## 11. PWA 说明

项目已接入 PWA 配置，支持添加到手机主屏幕。

Manifest 关键配置：
- name: `个人状态记录器`
- short_name: `状态记录`
- display: `standalone`
- start_url: `/`

图标路径：

```text
public/icons/icon-192.png
public/icons/icon-512.png
```

### iPhone 安装方式
1. 使用 Safari 打开应用；
2. 点击“分享”；
3. 选择“添加到主屏幕”。

### Android 安装方式
1. 使用 Chrome 打开应用；
2. 点击浏览器菜单；
3. 选择“添加到主屏幕”或“安装应用”。


```bash

```

## 13. 路由说明

当前项目未使用 React Router，而是采用内部 Tab 状态切换，因此部署到 Vercel 时通常不需要额外 rewrite 配置。

## 14. 当前文件结构重点

```text

  constants/

  utils/
```
